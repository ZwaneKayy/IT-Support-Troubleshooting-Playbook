# IT Support Troubleshooting Playbook

This repository demonstrates hands-on IT support and entry-level cybersecurity skills through real-world troubleshooting scenarios and incident responses.

It reflects how issues are documented and resolved in professional helpdesk and IT operations environments.

## Skills Demonstrated
- IT support and helpdesk troubleshooting
- Windows operating system support
- Networking fundamentals (TCP/IP, DNS, DHCP)
- User account and access management
- Basic cybersecurity incident response
- Clear technical documentation

## Tools & Technologies
- Windows 10/11
- Command Prompt & PowerShell
- Networking utilities (ipconfig, ping)
- Basic security best practices
- Virtual lab environments

## Project Purpose
This project was created to demonstrate practical IT support experience and a strong foundation for roles such as:
- IT Support Technician
- Help Desk Analyst
- NOC Technician
- Junior SOC Analyst

## Career Goal
To progress into a Cyber Security Analyst role by building strong foundations in IT support, networking, and security operations.
